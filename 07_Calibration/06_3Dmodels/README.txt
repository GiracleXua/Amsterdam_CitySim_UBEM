# File name instruction:
The subdirectories include raw 3D models (not CitySim.xml format) of three sites in Amsterdam-Oost
- AOI_small
- AOI_medium
- AIO_large

Additional info. associated with the file name include:
- nonpolygon: if the the check box of the "export planer region as polygon" is checked when exporting citymodel from Rhino to .SKP (recommended)
- cleaned: if the building geometry is further simplified (removing triangulated polygons) in SketchUp

# NotUsed folder: can be deleted when the project is done
